Workflow - Lookdev

Do's

  • Make sure that the project is always set for the Maya scene that you are working on.
    • This makes sure that paths used stay relative, enabling reuse on other machines and operating systems.
  • Include the turn table light rig as a reference instead of adding it to the lookdev file.
    • This makes cleanup easier by just removing the reference.
    • Always remember to remove the reference before passing on the file downstream.

Don'ts

  • Do not use version names in texture files. Use versions only at the top-level directory outside the Maya project.

Correct:

                v0001
                |_maya
                  |_sourceimages
                    |_baryonix
                      |_hi
                        |_exr
                          |_baryonix_col.1001.exe

Incorrect:


                maya
                |_sourceimages
                  |_baryonix
                    |_v0001
                      |_exr
                        |_ baryonix_col_v0001.1001.exr
