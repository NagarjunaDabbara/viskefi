Studio Pipeline

The 3D Animation Studio Pipeline is a detailed process to convert an idea into a finished product. It is a system designed to take each step in a pre-determined order so that everyone knows.

The Studio Pipeline goes through three major phases:

  • Pre-Production
  • Production
  • Post-Production
