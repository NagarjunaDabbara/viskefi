Review Workflow - for Supervisors

ShotGrid Create Installation

ShotGrid Create helps supervisors/artists do their creative work with minimal distraction. To get started, download and install ShotGrid Create from https://viskefi.shotgunstudio.com/page/create_download.

! If you don't have access to dowload or install the application please contact IT department.

Login to ShotGrid Create

After the installation you can proceed to login into Shotgrid Create by providing the Shotgrid URL and Autodesk credentials.

Step-1 Provide Shotgrid URL https://viskefi.shotgunstudio.com/

Step-2 Provide Autodesk Credentials to login into ShotGrid Create.

Review In ShotGrid Create

ShotGrid Create helps supervisors to review the work that artists submit. To start reviewing navigate to My reviews tab from the dropdown on top left corner.

Next select any one of the listed pending reviews to review.

Compare two versions side by side as shown below.

Annotate and record your comment and hit Send Approval to submit the review notes

Previous notes can be viewed by clicking Info the button on top right corner.

You can also refer the Review work flow for artists documentation for further information.
