Template

CreateLights

Description:

  • This tool create lights with predefined settings.

Tools:

  • Directional Light - Create Directional Light.
  • Point Light - Create Point Light.
  • Spot Light - Create Spot Light.
  • Skydome Light - Create Skydome Light.
  • Area Light - Create Area Light.
  • Softbox Area Light - Create Area Light with Softbox map connected in Light color.
  • Light Controller - Select Light/Lights and click Light Controller. It will create a Locator, which will work as a Light Target.

Light Attributes

Description:

  • This tool create lights with predefined settings.

Tools:

  • Use Color Temperature - This will Tick Use Color Temperature Attribute for selected lights.
  • Color - If Use Color Temperature is unchecked, it will open Maya Color Picker. If Use Color Temperature is checked, it will only show Color Temperature in the swatch.
  • Temperature - Change Color Temperature for selected lights. Make sure Use Color Temperature is Ticked.
  • Exposure - Change Light Exposure for selected lights.
  • Samples - Chnage Light Samples for selected lights.
  • Light Group Name - Add Light Group Name for selected lights.

Texture Colorspace

Description:

  • This tool will set Colorspace for selected file/aiImage nodes. You can also select textures from TX Manager.

Tools:

  • ACES - ACEScg = Set Colorspace to ACES - ACEScg.
  • Utility - Raw = Set Colorspace to Utility - Raw.
  • Utility - Linear = sRGB - Set Colorspace to Utility - Linear - sRGB.
  • Utility - Linear = sRGB - Set Colorspace to Utility - Linear - sRGB.
  • Utility - Linear = sRGB - Set Colorspace to Utility - Linear - sRGB.
  • Utility - Linear = sRGB - Set Colorspace to Utility - Linear - sRGB.
  • Utility - sRGB = Texture - Set Colorspace to Utility - sRGB - Texture.
  • Output - sRGB = Set Colorspace to Output - sRGB.
  • Output - Rec.709 = Set Colorspace to Output - Rec.709.

Shape Attributes

Description:

  • This tool will change Shape Attributes of selected Geometries or Selection Sets.

Tools:

  • Primary Visibility - Sets On or Off Primary Visibility.
  • Casts Shadows - Sets On or Off Casts Shadows.
  • Self Shadows - Sets On or Off Self Shadows.
  • Diffuse Reflection - Sets On or Off Diffuse Reflection.
  • Specular Reflection - Sets On or Off Specular Reflection
  • Diffuse Transmission - Sets On or Off Diffuse Transmission
  • Specular Transmission - Sets On or Off Specular Transmission
  • Matte - Sets On or Off Matte
  • Catclark - Sets On or Off Catclark Subdivision.
  • Catclark Iterations - Sets Catclark Subdivision Iterations.

Light Filters

Description:

  • This tool will allow you to Create, Disconnect or Delete Arnold Light Filters from Maya Scene. This will work on multiple selected lights.

Tools:

    Connect:

  • Dropdown List - A Dropdown List of all Arnold Light Filters exists in your current Maya Scene.
  • Reload - This will Reload/Update Dropdown List. This step is required if any Arnold Light Filter is created manually.
  • Connect - This will connect selected Arnold Light Filter from Dropdown List to the selected lights.
  • Disconnect - This will disconnect selected Arnold Light Filter from Dropdown List to the selected lights.
  • Delete - This will delete selected Arnold Light Filter from current Maya Scene.

    Create:

  • Gobo - Create and Connect Gobo on selected lights.
  • Barndoor - Create and Connect Barndoor on selected lights.
  • Light Blocker - Create and Connect Light Blocker on selected lights.
  • Light Decay - Create and Connect Light Decay on selected lights.

Geometry Pref

Description:

  • This tool generates Pref (Position Reference/Texture Reference) on mentioned frame on selected Geometries.
  • After generating Pref, you can look for “mtoa_varying_Pref” attribute on shape node to confirm if it is generated, or just load Pref AOV and render.

Tools:

  • Frame - Frame number on which you want to generate Pref.
  • Generate - Generate Pref on mentioned frame on selected Geometries.
  • Delete - Delete Pref on selected Geometries.

Position Mask

Scene Scale

Description:

  • This tool will set selected Lights Exposure as per Scene Scale. Just select all the required lights, type Old and New Scale and click Set.

Tools:

  • Old Scale - Old Maya Scene Scale.
  • New Scale - New Maya Scene Scale.
  • Set - Set Light Exposure as per Old and New Scale.

Camera Overscan

Description:

  • This tool will set Camera Overscan for selected camera and scene image resolution.

Tools:

  • Old Resolution - Current Maya Scene Image Resolution.
  • New Resolution - New Maya Scene Image Resolution as per Camera Overscan value.
  • Reverse - Reverse will reverse the Camera Overscan calculation. This is useful to revert back Camera Overscan. For example, if you set overscan to 1.2, and you want to remove the overscan, then just tick Reverse and type 1.2 overscan again.
  • Camera Overscan - Camera Overscan value.

Camera Split

Description:

  • This tool will split frame to pieces. There is one fixed Vertical Split and you can specify Horizontal Splits.

Tools:

  • Horizontal Split - Number of Horizontal Splits for frame splitting.
  • Create Cameras - Create number of cameras as per Horizontal Split. You have to first select Render Camera.
  • Save Files - Select all Split Cameras and Save Files. Same number of files as cameras will be saved with Camera Suffix in the scene folder. Cameras will be set in their respective files.
