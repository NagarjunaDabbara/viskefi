Pipeline Tools - Usage

Pyblish :

  • Pyblish a publish tool included in Maya for various departments to publish the artists work to the upstream department.
  • How to launch : Maya > viskefi-menu > Load pyblish File > pyblish .

    • Scene Builder :

  • Scene builder is a tool designed for the building of the lighting scene.
  • How to launch : Maya > viskefi-menu > Load Scene builder .

    • Viskefi Toolbar (for lighting dept.):

  • Viskefi toolbar contains the tools that are useful in the scene building for lighting department.
  • How to launch : Maya > viskefi-menu > Load Viskefi Toolbar .
    • Viskefi Toolbar is Docked at the Right of the Maya below the attribute editor.

    Rig Builder :

  • Rig builder is a tool designed for the building of the Rig scene.
  • How to launch : Maya > viskefi-menu > rigging > Load rig Builder .

    • Sanity Checker :

  • Sanity checker a tool for the modeling department for checking the scene before publishing.
  • How to launch : Maya > viskefi-menu > modelling > Sanity Checker .
