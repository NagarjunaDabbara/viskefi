Install Autodesk Maya on Personal Workstation

Instructions

This tutorial assumes that you have already got access to your Autodesk credentials that uses @viskefi.com email. If not, raise a software access request at the technology helpdesk

Login to https://accounts.autodesk.com with your account credentials

  • Navigate to Products and Services


  • Click on View downloads to list the software available


  • Choose the pipeline recommended version of the software and select Install Now


  • Once installation is complete, sign in using the same credentials
