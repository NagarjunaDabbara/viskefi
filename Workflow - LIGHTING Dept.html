Workflow - Lighting Dept

Inputs

  • Lighting Dept inputs: For every Dept except COMP Dept.
  • Includes animation, assets, props, shaders, caches, and FX elements.
  • Rigging manual match shader used.
  • Lighting updates files from FX Dept through hidden restricted steps.
  • Animation updates: Delivered daily or as needed.
  • Caches and cameras are shared separately.
  • FX cache versions: Delivered with QA check.
  • Specific Inputs:
    • Animation: Maya cache updates.
    • Rigging: Rig updates.

System Requirements

  • Animation: Keyframe cache.
  • FX Input Assets:
    • VDB generation.
    • Mesh volumes.
    • Camera work.
    • Object tools.
  • Final layout animation.
