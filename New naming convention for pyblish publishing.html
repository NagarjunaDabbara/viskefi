New naming convention for pyblish publishing

Naming for Camera publish:

  • main camera group name : renderCam_grp
  • main camera name : renderCam

    • Naming for Camera mmgeo publish:

  • mmgeo name : mmGeo_grp

    • Naming for Model publish:

  • model naming convention : {assettype}{AssetName}_grp
  • model example name: chrFrog_grp

    • Naming for Rig publish:

  • requirements:
    1. set name: abcSet
    2. set contains: modelGrp group
    3. modelGrp group contains: Published Model
  • rig naming convention : {assettype}{AssetName}_rig
  • rig example name: chrFrog_grp
