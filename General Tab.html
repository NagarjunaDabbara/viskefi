General Tab

General Tab contains common tools, which will be used by multiple departments.

  • Prefix/Suffix
  • Description:
    1. This tool is useful to add Prefix or Suffix on selected Maya Nodes. It supports selection from Outliner and Node Editor/Hypershade.
  • How to use:
    1. Select nodes, type out Prefix or Suffix and press Enter on keyboard.
    2. if Underscore is checked, it will automatically add underscore. For example prefix_nodeName_suffix.
