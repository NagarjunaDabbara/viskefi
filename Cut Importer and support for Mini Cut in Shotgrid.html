Cut Importer

The Import Cut app is a simple and easy way to create a Cut in ShotGrid based on an EDL from editorial.

Import Cut will display the new in points, out points, and frame ranges against what was already tracked in ShotGrid, highlighting any differences.

Import Cut is built into RV and can be accessed via the "ShotGrid" menu or by Alt+I shortcut key in RV.

Steps to Create a Cut

  1. Export an EDL (CMX)

    You can get the .edl file from either the client or the editorial department.

    Sometimes you are provided with the .XML file instead of the .EDL. In this case, you can follow this documentation on how to convert .XML to .EDL. How to Convert .XML to .EDL
  2. Open RV and launch Import Cut

    Find it in the "ShotGrid" menu or by Alt+I shortcut in RV.

  3. Drag in the EDL (and Media, if available)
