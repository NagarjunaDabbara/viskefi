Workflow - ASSET Dept

Asset Dept = Modeling And Texturing

    INPUTS:

  • Asset dept collects reference images as .JPEG, .PNG.
  • In some cases client provides the photogrammetry mesh,Latter scan,CAD-mesh files.
  • Asset dept are first dept to start the workflow so Asset dept don't get any input files from any Dept.
  • Asset dept works using Maya and Z-Brush for modeling and Mari and substance for Texturing.
  • Texture Dept gets .obj and .ma file formats.

    OUTPUTS:

  • Asset dept send files to Rigging if Character exist,Lighting, FX and Groom dept.
  • Asset dept exports model,Textures and Lookdev(Shading) as one file.
  • Texture dept exports output files in .exr/.jpeg formats.
  • Only Groom dept gets .abc files and rest all output dept of Model dept exports .ma files.
