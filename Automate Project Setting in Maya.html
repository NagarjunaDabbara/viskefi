Automate Project Setting in Maya

When we are Lunching maya through shotgrid basic shot information should load.Frame Range, FPS, Colorspace,render settings & Resolution.

Frame Range:-

  • Launch the maya from shotgrid toolkit.
  • From shotgrid menu select the FileOpen.
  • Select the task and create a New File if it not created

    • From ShotGrid menu select the “Sync Frame Range from ShotGrid”.

    The frame range automatically get set to the file.
